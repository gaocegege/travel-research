# 旅游计划

## 候选目的地

### 瑞士

苏黎世，卢塞恩，少女峰，日内瓦等等比较受欢迎，特色食品奶酪火锅，多人评价食物并不好。

#### 费用

可乐 15元
青旅床位 250元
当地矿泉水 10元
当地啤酒 30元
一顿饭 150元

以上由百度提供，住得好的话，一天大约 600。百度给出的参考费用 15k / 10 天，住的估计并不好，所以真实费用大约 20k-30k / 10 天，不含机票。各个网站自由行费用大约机票 + 酒店 20k / 10 天。

PS: 阿姨在这里
